#  TROUBLESHOOTING
# ============================================================================

Issue: Skills not showing correct levels
Solution: Ensure resume text includes proficiency keywords
          (e.g., "Expert in Python", "Advanced AWS")

Issue: Work experience not parsing correctly
Solution: Check resume format has clear company/position lines
          Consider: "Position at Company" format

Issue: Education degree not recognized
Solution: Use standard degree names
          (Bachelor, Master, Ph.D., Associate)

Issue: Projects not detecting technologies
Solution: Verify technologies are written as recognized names
          Check list of 50+ supported techs

Issue: Profiles not extracting
Solution: Ensure LinkedIn/GitHub URLs are valid
          Format: linkedin.com/in/username
